วิธีการใช้งานเครื่องซัก

1. นำผ้าเข้าเครื่องซัก (เครื่องที่จะซัก) โดยที่หน้าเครื่องจะมีหมายเลขเครื่องระบุไว้

2. นำน้ำยาซักผ้า และน้ำยาปรับผ้านุ่มใส่ที่ช่องใส่น้ำยา (ช่องใส่น้ำยาจะอยู่บนเครื่องซักผ้า ให้เปิดฝาออกและใส่น้ำยาตามช่องที่ระบุไว้) กรณีใช้ผงซักฟอก ให้ใส่พร้อมผ้าที่จะซักในถังซักได้เลย

3. กดทำรายการที่เครื่องเวนดิ้ง โดยเลือกเมนูซัก จากนั้นเลือกเบอร์เครื่องซักที่ลูกค้าใส่ผ้าไว้ กดเลือกอุณหภูมิน้ำ กดชำระเงิน และกดเริ่มทำงาน ลูกค้าสามารถเลือกที่จะเริ่มซักทันที หรือไปกดเริ่มเองที่หน้าเครื่อง (กรณีต้องกลับไปใส่นํ้ายา)

4. ในกรณีที่ลูกค้ามีแอพพลิเคชันโค้ดคลีนแล้วให้กรอกเบอร์โทรศัพท์ลูกค้าและกดตกลงเพื่อรับการแจ้งเตือนจากแอพพลิเคชัน Code Clean ซึ่งจะแจ้งเตือนเมื่อผ้าซักเสร็จแล้ว

วิธีการใช้งานเครื่องอบ

1. นำผ้าเข้าเครื่องอบ โดยที่หน้าเครื่องจะมีหมายเลขเครื่อง ระบุบไว้

2.กดทำรายการที่เครื่องเวนดิ้ง โดยเลือกเมนูอบ จากนั้น เลือกเบอร์เครื่องอบที่ลูกค้าใส่ผ้าไว้ และกดเลือกอุณหภูมิ

3.กดชำระเงิน กดเริ่มทำงาน ลูกค้าสามารถเลือกที่จะเริ่ม อบทันที หรือไปกดเริ่มเองที่หน้าเครื่อง (กรณีต้องกลับไป ใส่ผ้าในตู้)

4.ในกรณีที่ลูกค้ามีแอพพลิเคชันโค้ดคลื่นแล้ว ให้กรอก เบอร์โทรศัพท์ลูกค้าแล้วกดตกลง (แอฟพลิเคชันจะแจ้ง เตือนเมื่อผ้าอบเสร็จแล้ว)
